
The mistakes I made in learning Vue.js

1.el:'#example',  /*need '#'  !!!*/

2.el:'#ele-1'
    NOT
    ele:'#ele-1'

3.{{ msg }} should be an attribute of vue instance's data attribute:
    RIGHT:
        ...
        data: {
            msg:"I love you~"
        }
    WRONG:
        ...
        msg:"I love you~"
        ...

4.Control data object need "$":
     app3.$data.dynamicClass="font-green";

5.If a function is not defined,the error will be remind:
    vue.js:525 [Vue warn]: Error when rendering root instance:
    ...
    vue.js:2229 Uncaught ReferenceError: conLog is not defined
